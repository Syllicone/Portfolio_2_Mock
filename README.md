# danielsinclair.design

Personal portfolio site built with plain HTML, CSS, and JavaScript.

Live at [danielsinclair.design](https://www.danielsinclair.design)

\---

## Design

Inspired by HTML brutalism and terminal interfaces. The aesthetic lends itself to feel more like a tool. The site is both for me and its viewers.

The navigation is open-ended by design, making it easy to slot in new sections and content without restructuring the layout.

\---

## Structure

```
/
├── About
└── Work
    ├── Projects
    ├── Writing
    ├── Photography
└── Links                        # in development         

└── Settings                     # in development

└── Tools                        # in development    

&#x20;   ├── Image processing
    └── QR code generation
```

\---

## Sections

**About** — Bio, CV, Photo, Qualifications, Contact, What's in my pencil case

**Work** — A collection of creative and technical output across four areas:

* *Projects* — Design work
* *Writing* — Essays and publications
* *Photography* — Ongoing photo series of various topics

**Tools** — In-browser utilities, no dependencies, no accounts. Planned tools include basic image processing and QR code generation.

**Settings** — User-facing controls to customise the site's appearance and behaviour to their own preference.
